TRANSACTION API

POST 서명된 메시지 전송
https://api.allbit.com/open/v2/submit-signed-message

주문 등록 및 취소를 위해 생성 및 서명된 메시지를 전송한다.

  • 서명 방법

    1. 주문 등록/취소 API를 통해 actionHash 조회, action을 요청시 입력
    2. web3 같이 Ethereum Util을 이용해 서명 (\x19Ethereum Signed Message:\32가 prefix여야 함.)
    3. 서명 후 결과로 출력되는 v, r, s 값을 요청시 입력

  • Example (node.js)

      let privateKey = '0x4c0883a69102937d6231471b5dbb6204fe5129617082792ae468d01a3f362311';
  let action = 1;
  let actionHash = '0x000713e6e94042585c348b38b00ef810ae1756e38e04cc01e6329a8334f05ea5';

  let sig = web3.eth.accounts.sign(actionHash, privateKey);

  let msg = {};
  msg.action = action;
  msg.actionHash = actionHash;
  msg.message = {
      v: web3.utils.toHex(sig.v),
      r: sig.r,
      s: sig.s
  }
HEADERS
Content-Type
application/json
BODY
{
  "apiKey": 발급받은 API Key,
  "wallet": Allbit 지갑주소,
  "messages": [
    {
      "actionHash": 서명할 메시지,
      "action": Action ID,
      "message": {
        "v": 서명 결과 V값 (1byte hex),
        "r": 서명 결과 R값 (32byte hex),
        "s": 서명 결과 S값 (32byte hex)
      }
    }
  ]
}


Example Request

  \"wallet\": \"0xfaa8ed1fbac594dd017121df8ebd4fe2bea9016f\",
