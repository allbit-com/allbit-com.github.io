TRANSACTION API

POST Submit signed message
https://api.allbit.com/open/v2/submit-signed-message

Sends generated and signed messages for order registration and cancellation.

  • How to sign
    1. By Order registration/cancellation API, Put actionHash, action
    2. Sign with Ethereum utils, such as web3. (\x19Ethereum Signed Message:\32is should be prefix.)
    3. Get v, r, s value by signing, and put them when get request.

  • Example (node.js)

      let privateKey = '0x4c0883a69102937d6231471b5dbb6204fe5129617082792ae468d01a3f362311';
  let action = 1;
  let actionHash = '0x000713e6e94042585c348b38b00ef810ae1756e38e04cc01e6329a8334f05ea5';

  let sig = web3.eth.accounts.sign(actionHash, privateKey);

  let msg = {};
  msg.action = action;
  msg.actionHash = actionHash;
  msg.message = {
      v: web3.utils.toHex(sig.v),
      r: sig.r,
      s: sig.s
  }
HEADERS
Content-Type
application/json
BODY
{
    "apiKey": Generated API Key,
    "wallet": Allbit Wallet Address,
    "messages": [
        {
            "actionHash": message to sign,
            "action": Action ID,
            "message": {
                "v": V value of signature result (1byte hex),
                "r": R value of signature result (32byte hex),
                "s": S value of signature result (32byte hex)
            }
        }
    ]
}
